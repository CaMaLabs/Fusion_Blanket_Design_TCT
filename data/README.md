# Data Directory

Use this directory for curated, shareable datasets only.

Recommended contents:

- CSV files needed to reproduce public figures
- JSON configuration files for selected runs
- small metadata files describing provenance

Do not place large raw dumps, duplicate zip bundles, private notes, or unreviewed exploratory outputs here. Put those in `archive/` or keep them local until reviewed.

Every dataset promoted here should include:

- source script or notebook
- date generated
- parameter set
- units
- meaning of each column
- whether values are exact simulation outputs or postprocessed summaries
