# Provenance

This repository is maintained by **Chase Lunsford / @chaseakat** as a public timestamped research record.

The commit history, documentation, simulation scripts, candidate structures, validation notes, result files, and related artifacts are part of the public development record for the fusion blanket / TCT concept path.

## Purpose

This repository was made public to preserve the development path, assumptions, code, candidate designs, and validation workflow in a visible form.

The work should be read as independent computational research unless a specific result is explicitly marked as externally validated.

## Scope of record

The provenance record includes:

- source code,

- committed candidate geometries,
- optimizer structure,
- simulation parameters,
- validation scripts,
- result summaries,
- issue and pull request discussions,
- and timestamped commit history.

## Validation status

This repository includes exploratory and hypothesis-level work. Claims should be interpreted according to the validation level documented in `docs/TCT_Validation_Matrix.md`.

Unless otherwise stated, repository outputs are screening results and should not be treated as finished reactor engineering claims.
