# Topology Hypotheses

## Tested stacks

### 1. Be sandwich
Be → Li2O → Li2O → Be → W composite

Strong breeder, poor attenuation.

### 2. Outer kill layer
Be → Li2O → Li2O → W composite → Be

Attempts to contain leakage at boundary.

### 3. Center kill
Be → Li2O → W composite → Li2O → Be

Controls energy cascade mid-blanket.

## PbLi variants (new)

### PbLi front multiplier
PbLi → Li2O → Li2O → W composite → Be

Hypothesis: increase neutron population before breeding.

### PbLi + shaping
PbLi → Li2O → W composite → Li2O → Be

Hypothesis: balance multiplication and shaping earlier.

### Liquid + PbLi hybrid
Li wall → PbLi → Li2O → shaping → tail

Still experimental.

## What we are testing

- Does early multiplication improve total usable neutron population?
- Does PbLi degrade spectrum too early?
- Where is the optimal location for absorber layers?
